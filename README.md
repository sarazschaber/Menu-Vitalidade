# Menu Vitalidade

Um projeto frontend simples que apresenta um cardápio nutricional com foco em opções saudáveis e proteicas. Ideal para mostrar um menu de refeições organizado por tipo e dia da semana.

<h2>Visão geral</h2>

A página index.html exibe um layout limpo e acolhedor, com:

- Título e descrição da proposta nutricional;
- Botões de navegação para os tipos de refeição: café da manhã, almoço, lanche e jantar;
- Filtro de dias da semana para visualização segmentada;
- Opções de refeições com informações de ingredientes e tabelas de valores nutricionais;
- Design em tons de verde para transmitir saúde e bem-estar.

Arquivos
- "index.html": estrutura da página
- "cardápio.css": estilo e aparência do projeto

<h2>Tecnologias</h2>

- HTML5
- CSS3
